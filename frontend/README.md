# Todo Frontend

A modern Next.js frontend for the Todo application with AI chatbot integration.

## Features

- User authentication (signup/signin)
- Task management (create, read, update, delete)
- AI-powered chatbot assistant
- Responsive design

## Backend API

This frontend connects to the backend API hosted at:
https://saimawaheedsaima5-todo-backend-api.hf.space
